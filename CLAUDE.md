# Runner's Record 開発メモ

## プロジェクト概要
マラソン等大会結果記録・共有ツール
- Next.js + TypeScript + Tailwind CSS
- Supabase（認証・データベース）
- 知人の使用を想定

## ページ構成
- `/` - ホームページ（各ページへのリンクカード表示）
- `/latest` - 最近の記録（直近1年間）
- `/search` - 検索ページ
- `/links` - リンク集
- `/profile` - プロフィール（ログイン必要）
- `/records` - 自分の記録（ログイン必要）
- `/records/form` - フォームで記録追加（ログイン必要）
- `/records/upload` - CSV形式で記録追加（ログイン必要）
- `/records/edit` - 記録を編集（ログイン必要）
